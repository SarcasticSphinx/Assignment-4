1. What are the different ways to select an element in the DOM?

You can select elements using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

2. What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets or sets the HTML content of an element.

innerText: Gets or sets the visible text, excluding hidden elements.

textContent: Gets or sets the text, including hidden elements, without formatting.

3. What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its children using event bubbling. This is useful for dynamically added elements.

4. What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts from the target element and propagates up to its ancestors, allowing handlers on parent elements to react.

5. How do you create, add, and remove elements using JavaScript?

You can create an element using document.createElement(), add it using appendChild() or insertBefore(), and remove it using removeChild() or remove().
